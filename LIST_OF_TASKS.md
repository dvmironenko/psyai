## Приоритет 2: FastAPI backend

### Задача 2.1: Конфигурация FastAPI
**Приоритет:** высокий
**Зависимости:** 1.1

- `app/config.py` - классы конфигурации (DevelopmentConfig, ProductionConfig)
- `app/database.py` - engine и SessionLocal для SQLAlchemy
- `app/main.py` - главная точка входа FastAPI

**Зависимости:**
- Базовые зависимости из requirements.txt

---

### Задача 2.2: Модели и схемы Pydantic
**Приоритет:** высокий
**Зависимости:** 2.1

- `app/models.py` - модель User (id, email, password_hash, is_active, is_admin, plan_id, created_at)
- `app/models.py` - модель UserPlan (id, name, description)
- `app/schemas.py` - Pydantic схемы для валидации

---

### Задача 2.3: Аутентификация
**Приоритет:** высокий
**Зависимости:** 2.1, 2.2

- `app/auth.py` - OAuth2PasswordBearer
- Функции хеширования паролей через bcrypt (passlib)
- Токены доступа сроком 30 минут

---

### Задача 2.4: FastAPI роутеры
**Приоритет:** высокий
**Зависимости:** 2.3

- `app/routers/auth.py` - `/login`, `/register`, `/reset-password`
- `app/routers/users.py` - `/me`, `/me/profile`, `/deactivate`
- `app/routers/admin.py` - CRUD для users/plans, `/stats`
